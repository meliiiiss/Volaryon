# 🌀 Volaryon RPG: Legado dos Deuses

**Volaryon** é um sistema de RPG de mesa autoral que equilibra a profundidade tática de *Pathfinder 2e* com a fluidez narrativa de *D&D 5e*. Ambientado num mundo onde o destino é moldado por entidades primordiais, os jogadores assumem o papel de seres marcados pela essência divina.

## 📂 Estrutura do Repositório

O projeto está organizado para facilitar a automação e a consulta rápida:

* **`/js`**: Lógica do sistema (automação de fichas, base de dados de deuses e calculadoras).
* **`/css`**: Estilização visual (layouts luxuosos inspirados em *Legacy of Gods*).
* **`/ativos`**: Recursos de imersão (ícones de habilidades, molduras decorativas e média).
* **`ficha.html`**: Interface interativa do jogador.
* **`manual.html`**: Guia de regras integrado.

---

## 🛠️ Pilares do Sistema

1.  **Evolução por Atributos**: Sistema de distribuição de 30 pontos sobre uma base de 5.
2.  **O Despertar (D25)**: Sorteio de uma das 25 entidades divinas que concede habilidades evolutivas.
3.  **Linhagens Complexas**: Combinação de Raça e Sub-raça com bônus passivos únicos.
4.  **Interface Progressiva**: Fichas dinâmicas que calculam CA e Modificadores em tempo real.

---

## 🚀 Como Contribuir

1.  **Mecânicas**: Sugira novos monstros no `Bestiário.md` ou habilidades no `Ideias e Conceitos.md`.
2.  **Código**: Melhore a automação da ficha em `js/script.js`.
3.  **Lore**: Expanda as regiões no `Atlas de Volaryon.md`.
